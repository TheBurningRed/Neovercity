### Висновки
1. **Точність методу**: Метод Монте-Карло дає результат, дуже близький до аналітичного. Однак цей метод є стохастичним (імовірнісним), тому результат дещо змінюється при кожному новому запуску.
2. **Залежність від кількості точок**: Точність методу Монте-Карло безпосередньо залежить від кількості випадкових точок N. Для високої точності потрібно дуже багато ітерацій.
3. **Порівняння**: Функція використовує чисельні методи інтегрування (наприклад, метод Квадратур), які є значно ефективнішими та точнішими для одновимірних функцій, ніж метод Монте-Карло. `quad`
4. **Практичне застосування**: Хоча для простих інтегралів цей метод програє у швидкості, він стає незамінним при обчисленні складних багатовимірних інтегралів, де традиційні сіткові методи стають занадто ресурсомісткими.
